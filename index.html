HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or atttributes to which the rules or declarations, will apply
  • Declaration(s) contain two important components:CSS property and value.
  • "*" or wild card symbol is applied a rule to all elements visible on page
  • The img selector assigns property values to any image element in the markup
  • When an external CSS file is created it must be linked to an HTML file.
  • Class Attributes allows us to share a CSS rule to any element we choose by assigning the rule to a class attribute with selector
  • A (.) before a selector (i.e card) designates a class selector
  • Any HTML element assigned to the class with gain the property assignments from CSS
  • Border propery assigns a border

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is simply a name container that allows us to store data in our code
  • A variable is declared using var
  • The assignment operator is used to assign specific data that we want stored in our var
  • Control Flow is the order in which a computer executes code in a script
  • Conditional Statements and loops control the order in which code is executed
